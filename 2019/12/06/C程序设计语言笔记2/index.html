C程序设计语言笔记2

发表于 2019-12-06 | 分类于 C程序设计语言笔记

1. 变量名

  • 变量名由字母和数字组成

  • 第一个字符必须是字母

  • “_“被看做字母

  • 大小写字母有区别

说明：

  1. 库中的名字常以”_“开头，因此最好不要用”_“。
  2. 传统C语言中，变量名使用小写字母，符号常量名全部使用大写字母
  3. if, else, int等关键词不能作为变量名
  4. 局部变量一般使用较短的变量名，外部变量使用较长的变量名（便于从名字上看出其用途）

2. 类型

2.1 数据类型及其长度

数据类型 说明 占用大小
char 字符型 8位
int 整型 16位，或者32位
float 单精度浮点 32位
double 双精度浮点 64位

2.1.1 long, short限定符

书中将long，short作为限定符，但只能限定int，因为不同机器int类型占用的空间不一样。short通常为16位，long通常为32位。

c语言规定：

  • short至少16位
  • long至少32位
  • short类型不能长于int
  • int类型不能长于long

一般在Windows下编程，short是16位，int是32位，long也是32位，long long是64

2.1.2 signed, unsigned限定符

  • 这两个用于限定整型和char类型
  • signed修饰的char是-128~127,
  • unsigned修饰的char是0~255
  • 不带修饰的char取决于机器

2.1.3 其他

  1. 一般float 32位，double 64位，long double 128位，但也可能三个占用长度一样（取决于机器）
  2. 这些类型长度的定义，在<limit.h>和<float.h>中有说明，这里就不展开了。
  3. c语言中是没有bool类型的，只有c++才有。但是，c有long long和_Bool（我测试GCC -std=c98也可以编译成功？）

2.2 常量

  1. long 常以字母L（大小写皆可）结尾。
  2. unsigned long以ul结尾，unsigned long long以ull结尾
  3. 以0开头的数字，表示8进制（0是数字不是字母）
  4. 以0x/0X开头的数字，表示16进制（同上）

常量表达式：只包含常量的表达式，在编译时求值，不在运行时求值

字符常量：本质是整型，每个字符对应其ASCII码

  • 空字符：’\0’，特殊字符，值为0，表示NULL

转义字符：看起来像多个字符，实际上只是一个

  • ‘\000’：000表示1~3位八进制数
  • ‘\xhh’：hh表示两位十六进制数

字符串常量：也叫字符串字面值

  • char s[] = "hello, " "world";这样可将多个字符串连接起来
  • strlen(char s[])返回字符串长度，但是不包括’\0’

枚举：枚举是一种特殊数据类型，放在这里说是因为其和常量值联系紧密

  • 定义：emnu boolean{NO, YES};
  • 没有显示说明，emnu类型第一个值为0，第二个为1，以此类推。
  • 如果指定了部分值，那么未指定的值为前一个的值+1
  • 同一类型中的枚举名字必须不同
  • 同一类型中不同名字的枚举值可以相同

2.3 声明

声明：变量类型+变量名，说明变量的类型和名字

初始化：变量名 = 值，将变量赋初值

  • 任何变量都可以用const修饰，表示其值无法再被改变

  • const也可配合数组参数使用

3. 运算符

注意不同运算符之间的优先级关系。

3.1 算数运算符

  • 取模”%”不能用于浮点型
  • 整数取模取决于机器
  • 负数取模可以单独写一篇博客了

3.2 关系运算符与逻辑运算符

|| 和 && 逻辑判断的特殊用途

  • a || b：若a为真，则b不进行计算
  • a && b：若a为假，则b不进行计算

下面看个例子

for循环中的判断表达式一共有三个部分

  • i < lim - 1
  • (c = getchar()) != '\n'
  • c != EOF

这三个部分由两个&&连接，编译器会从左往右计算每个表达式的值，如果遇到某一个值为假，则后面的语句就不会运行。例如i = 4, lim = 5时：

i < lim - 1 就是 4 < 5 - 1为假，则(c = getchar()) != '\n'也不会运行。

意思是已经读取到了限制范围的最大值了，那么就不应该继续往下读。

3.3 运算符优先级与求值次序

类别 运算符 结合性
后缀 () [] -> . ++ – 左->右
一元 + - ! ~ ++ -- (type) * & sizeof 右->左
乘除 * / % 左->右
加减 + - 左->右
移位 << >> 左->右
关系 < <= > >= 左->右
相等 == != 左->右
位与 AND & 左->右
位异或 XOR ^ 左->右
位或 OR | 左->右
逻辑与 AND && 左->右
逻辑或 OR || 左->右
条件 ? : 右->左
赋值 = += -= *= /= %= >>= <<= &= ^= |= 右->左
逗号 ， 左->右

大致记忆：

  • 括号，指针 > 一元运算符（正负号，自加减）> 四则运算 > 移位 > 位运算 > 逻辑运算 > 赋值运算
  • 在指针和一元运算符一起使用时要格外注意
  • 在位运算和其他运算一起使用时要格外注意

if ((x & MASK) == 0)不同于if (x & MASK == 0)，后者等价于if(x & (MASK == 0))

因为在不同编译器环境下表达式求值会不同（特别是使用自加自减运算符时），建议多使用”()”。避免招致不必要的bug。

求值顺序：

  • 同一运算符每个操作数求值顺序不定
  • 函数各个参数求值顺序不定

3.4 自增运算符和自减运算符

  • 前置自增/减：先增/减后，再使用值

  • 后置自增/减：先使用值后，再自增/减

  • 自增自减运算符只能作用于变量，无法作用于表达式

表达式a[i] = i++;a[i]引用新值还是旧值？

3.5 按位运算符

位运算符只能作用于整型操作数：char, short, int, long。有符号或者无符号皆可。

位运算非常巧妙，能实现很多意想不到的功能。需要在平时的编程中多积累，多思考。
